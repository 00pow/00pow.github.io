# GitHub 연동 기술 블로그

> 윈도우 환경에서 jekyll과 ruby를 이용한 jekyll 테마 적용된 GitHub 연동 블로그 생성 <br>
> 개인 사이드 프로젝트로, 오류 슈팅 및 개인 학습 내용을 기록하는 것을 목표로 하였습니다.

- 기간 : 2023.11.28 ~ 2023.11.30 (3일)
- 팀원 구성 : 김서영
- 기능 : 루비를 이용해 로컬 서버에 접속하여 블로그에 반영한 내용을 commit 전에 미리 확인 가능하다.
